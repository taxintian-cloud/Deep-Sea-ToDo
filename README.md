# 🌊 Deep Sea ToDo  

公開ページはこちら⇒https://taxintian-cloud.github.io/Deep-Sea-ToDo/

深海の静けさと光のゆらぎをテーマにした、幻想的な ToDo アプリです。  
タスク管理を「深海に沈める」ような感覚で楽しめる、世界観重視の作品です。

---

## 🐚 Features（主な機能）

### ✔ タスク管理
- タスクの追加 / 編集 / 削除  
- 完了アニメーション（深海に沈む “sinkFade” 演出）

### ✔ 深さカテゴリー
- 浅瀬（light）  
- 中層（middle）  
- 深海（deep）  
→ タスクの重要度や気分で選択できます

### ✔ 日付管理
- 期限付きタスク  
- 今日のタスクは光る  
- 期限切れタスクは暗い深海色に変化

### ✔ 繰り返し設定
- 毎日  
- 毎週  
- 毎月  
- 繰り返しなし

### ✔ 検索機能
- タスク名でリアルタイム絞り込み

### ✔ ドラッグ＆ドロップ
- タスクを自由に並び替え可能

### ✔ モーダルガイド
- 使い方をまとめたヘルプモーダル付き

### ✔ レスポンシブ対応
- スマホでも快適に操作できる UI  
- 入力エリアはスマホ時に縦並びへ最適化

---

## 🌌 Design Concept（デザインコンセプト）

Deep Sea ToDo は「深海の世界観」を UI 全体で表現しています。

- 多層レイヤーのバブルアニメーション  
- ガラス質感のカード（Glassmorphism）  
- 深海グラデーション  
- 光のゆらぎを感じるコアカラー  
- 静かに沈むアニメーション  

ただの ToDo アプリではなく、  
**“タスク管理という行為そのものを美しくする”**  
ことを目指した作品です。

## 🚀 How to Use（使い方）

1. タスクを入力  
2. 深さ・日付・繰り返しを設定  
3. 「追加」ボタンで登録  
4. タスクはクリックで完了  
5. ✏️ で編集、✕ で削除  
6. ドラッグで並び替え  
7. 右上の「？」でヘルプ表示

## 🔧 今後の改善ポイント

- **ローカルストレージの拡張**  
  タスクのバックアップ機能や、複数デバイス間での同期を検討。

- **タグ機能の追加**  
  深さカテゴリーとは別に、自由にタグを付けて整理できるようにする。

- **フィルター機能の強化**  
  「期限順」「深さ順」「未完了のみ」などの並び替えを追加。

- **アニメーションの軽量化**  
  モバイル環境でのパフォーマンス最適化を実施。

- **アクセシビリティ向上**  
  キーボード操作対応、ARIA 属性の追加など。

- **テーマ切り替え機能**  
  “深海”以外にも、浅瀬・夜光・氷海などのテーマを追加する構想。

- **PWA 対応**  
  スマホでアプリのように使えるインストール機能を検討。

- **タスクの詳細モーダル**  
  メモ欄やサブタスクを追加できる詳細画面を実装予定。
